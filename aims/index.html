МЕНЮ

Цель лабораторной работы:

ознакомление с принципом цифровой модуляции (манипуляции) сигналов, изучение на практике процесса манипуляции.

Порядок выполнения:

  1. Изучить принцип манипуляция сигнала.
  2. Понять разницу между видами цифровой модуляции.
  3. На практике осуществить преобразование сигнала: составить схему устройства и получить на экране осциллографа сигнал, менять его параметры для лучшего усвоения темы.
  4. Ответить на контрольные вопросы.

Справочная литература:

  1. Першин, В. Т. Основы радиоэлектроники : учебное пособие / В. Т. Першин. — Минск : Вышэйшая школа, 2006. — 399 c.
  2. Дашенков В.М. Исследование процессов амплитудной модуляции. - Мн.: БГУИР. 2001. - 21 стр.
  3. Андреев В. С. Теория нелинейных электрических цепей: Учебное пособие для вузов. — М.: Радио и связь, 1982 — 280 с.
  4. Баскаков С.И. Радиотехнические цепи и сигналы. - М.: Высш. школа, 1988.
  5. Кушнир В.Ф., Ферсман Б.А. Теория нелинейных электрических цепей. - М.: Связь, 1974.

Содержание отчета:

  1. Цель работы.
  2. Теоретическое обоснование.
  3. График полученного сигнала при амплитудной манипуляции.
  4. График полученного сигнала при частотной манипуляции.
  5. График полученного сигнала при фазовой манипуляции.
  6. Анализ полученных данных.
  7. Ответ на контрольные вопросы.
  8. Вывод.
